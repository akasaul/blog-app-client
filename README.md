# Full Stack Blogging site built with React, NestJS, SQL 

## Features

- Create a Blog Post ✍️
- Add Posts to Favorites ❤
- Search Posts 🔍
- Edit Posts (only Authenticated) ✍️
- Users Profile 👥
- Give Reaction (Multiple reactions) 🔥 🌟 💗
- Sign in and Sign out 👤
- Comments 🌟
